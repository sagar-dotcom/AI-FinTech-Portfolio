# Master Finance Calculator 🧮

A fully functional Command Line Interface (CLI) Finance Calculator built in Python. 

## 🚀 Features
1. **Basic Math:** Perform addition, subtraction, multiplication, and safe division.
2. **EMI Calculator:** Calculate monthly EMI, total interest, and total payment for loans.
3. **Bill Splitter:** Calculate tips and evenly split bills among multiple people with smart input validation.
4. **Tax / GST Calculator:**  Calculate GST amounts and total inclusive prices.

## 🛠️ Prerequisites
* Python 3.x installed on your system.

## 💻 How to Run
1. Clone this repository to your local machine.
2. Open your terminal or command prompt.
3. Run the script using the following command:
   ```bash
   python calculator.py
